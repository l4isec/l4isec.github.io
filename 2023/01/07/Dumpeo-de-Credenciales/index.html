Dumpeo de Credenciales

Hashes NTLM

sekurlsa::logonpassword es famoso por dumpear las contraseñas en texto claro de la memoria. Con los hashes NTLM podemos realizar un Pass The Hash o simplemente podemos crackear y conseguir una contraseña en texto claro.

mimikatz.exe sekurlsa::logonpasswords

Kerberos Encription Keys

sekurlsa::ekeys nos permite poder dumpear las keys de encription de kerberos de todo los usuarios que estan actualmente logueados. 

mimikatz.exe sekurlsa::ekeys

En este caso el hash AES256 es la que tenemos que guardar.

Security Account Manager

O mejor conocido como la SAM es una base de datos donde va estar alojado todos los hashes NTLM de las cuentas locales podemos dumpearlo con lsadump::sam.

mimikatz.exe lsadump::sam

Domain Cached Credentials

O mejor también conocidas como DCC, son hashes NTLM que no se pueden usar como para el Pass The Hash, la única forma de aprovechar esto es crackearlo en offline. lsadump::cache , también puede ser extraidos del HKLM\SECURITY

mimikatz.exe lsadump::cache

Extraer Kerberos Tickets

Podemos ultilizar Rubeus para llevar acabo esto, con el comando triage esto requiere de unos privilegios altos, lo que hace este comando es listar todo los tickets en la sesión que estamos logueada.

rubeus.exe triage

El output de esto nos lo va mostrar como algo así

LUID UserName Service EndTime
0x7049f jking @ web.NCORP.IO krbtgt/dev.NCORP.IO 9/1/2022 5:29:20 PM

Una vez que tengamos listado los tickets en este caso nos interesa el de jking, podemos extraerlo con dump lo que nos va extraer es el TGT.

rubeus.exe dump /luid:0x7049f /service:krbtgt /nowrap

DCSync

El DCSync se aprovecha del protocolo Directory Replication Service para extraer usernames y credenciales del DC.

Esto requiere de unos privilegios de GetNCChanges que acutalmente solo estará disponible para los Domain Admins.

Para hacer dcsync podemos usar directamente el comando que viene en el Cobalt Strike o lsadump::dcsync.

mimikatz.exe lsadump::dcsync /domain:dev.ncorp.io /user:DEV\Krbtgt
